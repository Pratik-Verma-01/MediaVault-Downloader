# MediaVault Downloader 🚀

A hyper-modern, aesthetically pleasing Social Media Video Downloader web interface. Designed with a strict focus on visual elegance, featuring an AMOLED dark theme and sleek Glassmorphism components.

## ✨ Features

- **Hyper-Modern UI:** Built with a deep AMOLED black background (`#050505`) to reduce eye strain and provide a premium feel.
- **Glassmorphism Design:** Beautiful frosted glass containers with high background blur, subtle luminous borders, and depth-enhancing shadows.
- **Dynamic Platform Detection:** Intelligently recognizes links from major platforms (YouTube, Instagram, TikTok) and adapts the UI accordingly.
- **Crisp Vector Graphics:** Uses exclusively high-quality SVG icons (Lucide React) for a sharp, professional look—no emojis.
- **Smooth Animations:** Features glowing neon cyan accents (`#00F2FE`) and seamless state transitions during the link analysis phase.

## 🛠️ Tech Stack

- **Frontend Builder:** Lovable 
- **Icons:** Lucide React (SVG-based)
- **Styling:** CSS/Tailwind (Glassmorphism & AMOLED Dark Mode themes)
